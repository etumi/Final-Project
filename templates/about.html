About

This Project is a machine learning project that aims to find the genre of the movie based on its plot. This is a multilabel text classification problem. The model takes the plot as an input and returns an array of genres. The selected genres used in the model are drama, comedy, action, thriller, animation, fantasy, romance, adventure, horror, crime. We used Natural Language Procesing (NLP) to convert the movie plots to meaningful input in our model and encoded the output genres. Our classification model was built using keras. Ou keras model had 2 Dense layers with a 10000 nodes in the first layer and 10 in the second. The accuracy of our model was 0.575. This project is was produced using the various languages we have learnt throughout this course.
